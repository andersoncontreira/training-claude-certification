# CLAUDE.md

This file provides context and conventions for Claude Code when working in this repository.

## Project Purpose

Study repository for the **Claude Certified Architect (CCA) — Foundations** certification exam.
Organized by the 5 official exam domains, with notes, exercises, and references.

## Repository Structure

```
/domain1-agentic-architecture/   27% of exam
/domain2-mcp-tool-design/        18% of exam
/domain3-claude-code/            20% of exam
/domain4-prompt-engineering/     20% of exam
/domain5-context-reliability/    15% of exam
```

Each domain contains:
- `README.md` — concept summaries and key notes
- `resources.md` — official links and references
- `exercises/` — progressive practical exercises (01, 02, 03...)

## Conventions

- All documentation is written in **English**
- Exercise files are numbered with zero-padded prefix: `01-`, `02-`, `03-`
- Code examples use Python unless another language is more appropriate for the topic
- External links should point to official Anthropic documentation when available

## When Helping With This Repo

- Exercises should be practical and progressively harder within each domain
- Prefer concrete code examples over abstract descriptions
- When adding resources, prioritize official Anthropic docs over third-party sources
- Keep README files concise — use bullet points and tables, not long prose

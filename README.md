## 12-Week Study Roadmap

### Foundation (Weeks 1–2): Prompt Engineering
Start here — everything else builds on strong prompting skills.

- [ ] Week 1: System prompts, XML tags, structured outputs
- [ ] Week 2: Chain-of-thought, few-shot, temperature & sampling

**Domain:** [domain4-prompt-engineering/](./domain4-prompt-engineering/)

---

### Core (Weeks 3–6): Agentic Architecture
The highest-weight domain — invest the most time here.

- [ ] Week 3: Agent loop, tool use, basic orchestration
- [ ] Week 4: Multi-agent patterns, subagent delegation
- [ ] Week 5: Human-in-the-loop, approval flows
- [ ] Week 6: Error handling, retries, agent reliability

**Domain:** [domain1-agentic-architecture/](./domain1-agentic-architecture/)

---

### Tooling (Weeks 7–8): Claude Code
- [ ] Week 7: CLI basics, hooks (pre/post), permissions model
- [ ] Week 8: Custom slash commands, multi-agent workflows in IDE

**Domain:** [domain3-claude-code/](./domain3-claude-code/)

---

### Integration (Weeks 9–10): MCP & Tool Design
- [ ] Week 9: MCP protocol, server/client lifecycle, tool schemas
- [ ] Week 10: Advanced tool design, security considerations

**Domain:** [domain2-mcp-tool-design/](./domain2-mcp-tool-design/)

---

### Reliability (Week 11): Context & Reliability
- [ ] Week 11: Context window limits, memory strategies, fallbacks

**Domain:** [domain5-context-reliability/](./domain5-context-reliability/)

---

### Review (Week 12): Full Revision
- [ ] Re-read all domain READMEs
- [ ] Redo all exercises from memory
- [ ] Practice with mock questions
- [ ] Review weak areas
